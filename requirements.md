# 시대인재 요구사항

## 요구사항

- 처음 알바 시작하는 사람을 위한 가이드
- 알바 일일 스케줄 표 생성
- 추후에 스케줄 시작 전 알림

  [한 세트 구성]

    - 입실: 수업 시작 시간 25분 전에 가 있어야 함 - 25분간 입실 + 라이브러리 배부
    - 퇴실: 퇴실 자료가 있는 경우, 퇴실 10분 전에 가서 대기 / 없는 경우 끝나고 가서 청소만 하면 됨
    - 조그: 수업 시작 5분 전 착석해서 수업이 끝날 때까지(100분) 녹화 및 인코딩 작업

  ⇒ 인원수가 부족한 경우, 동일한 시간대에 두 개의 강의가 있는 경우 → 한 명이 잠시 두 개를 봐준다거나 해도 되기 때문에

  **시나리오**

    ```jsx
    시작 전 (오전 크루만)
    - 수업 시작 전에 (강사 대기실 세팅, 조그 세팅, 수업 있는 모든 교실 세팅, 각 수업 자료들 체크)
    - 3명
      - 1, 2: 603호 세팅 + 조그 세팅 + 강사 대기실 세팅
      - 2, 3: 604호 세팅 + 물지우개/ 지우개 모든 교시 세팅 + 현판 세팅(강의실 안내판)
      - 3, 1: 605호 세팅 + 조식(스타벅스) 사와서 세팅 
      + 여기서 만약에 교실이 인원수보다 많으면 번갈아가면서 두 개 세팅 
    --------------------1교시랑은 독립시행 ------------------------------
    📍1교시 (8:30 ~ 10:10)
    604호 
    
    * 1교시 입실, 조그는 오전 크루만 가능 *
    * 1교시 퇴실부터 오후 크루 참여 가능 *
    
    입실: 장강민 (8:00 ~ 8:30) 입실(안내)
    						(8:40 ~ 9:20) 결석/지각자 자료를 독서실에 배부 
    조그: 김수민 (8시 25분 전에 컴퓨터 앞에 앉음 -> 끝날 때까지 조그로 이동하며 녹화. 이 때 9시 전까지는 조그 두 개 봄)
    퇴실: 문승찬 (10시 전에 미리 가서 대기, 만약 퇴실자료가 있다면 배부 후 칠판 정리 및 다음 수업 세팅)
    
    605호 
    입실: 조하은 (8시부터 대기 -> 8시 10분 ~ 8시 30분까지 입실 -> 결석/지각자 자료를 독서실에 배부) 
    조그: 조하은 (독서실에 배부 이후부터 조그 녹화: 8시 10분 ~ 9시까지는 입실+배부를 다 하고 이후부터 10시 10분까지 조그)
    퇴실: 임혜령 (10시 전에 미리 가서 대기, 만약 퇴실자료가 있다면 배부 후 칠판 정리 및 다음 수업 세팅)
    
    <- 미들(2명) (오전 9시에 출근해서 오후 5시까지 있음)
    
    📍2교시 (10:30 ~ 12: 10)
    - 2교시는 오전+미들 모두가 공평하게 count 제도
    604호
    입실: 김수민
    조그: 문승찬
    
    📍3교시 (13:20 ~ 15:00)
    - 3교시는 오전 + 미들 모두가 공평하게 count 제도
    - 오후 3명이 퇴실 담당, 입실 조그는 오전 + 미들 count로 
    604호
    입실: 김수민
    조그: 장강민
    
    606호
    입실: 문승찬
    조그: 조하은
    
    <- 오후(3명)  (오후 2시20분에 출근해서 10시까지)
    
    📍4교시 (15:20~ 17:00)
    <- 오전 퇴근 (미들 + 오후)
    
    ** 시험 있는 수업이라면? (나중 우선 순위)
    시후(보통 2명 필요): 김수민, 장강민(시험: 수업 시작 시간부터 30분)
    소요 시간 : 20 ~ 30분
    ```

  ## 요구사항 우선순위

  ### 1. 스케줄링

    - 오전 수업 전 스케줄링 (오전 count 기반)
    - 1교시 스케줄링 - 입실, 조그 (오전 count기반),  퇴실(오전, 미들 count 기반)
    - 2교시 스케줄링 - 입실, 조그 ,퇴실 (오전, 미들 count 기반)
    - 3교시 스케줄링 - 입실, 조그 (오전, 미들 count 기반) , 퇴실(오후 count 기반)

  ### 2.  시험 예외

    - 해당 교시 퇴실이 담당

  ### 3. 작업 교환

    - 작업을 교환하는 사람들끼리 합의 보면, 작업을 변경하고, count도 그에 맞춰서 수정
        - 어짜피 우선순위로 다음에 먼저 해당 작업 하게 될 확률 높아짐
